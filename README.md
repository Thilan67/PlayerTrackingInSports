# PlayerTrackingInSports
This project is a sports player detection system built using YOLOv8. It focuses on identifying players in sports videos and tracking their presence across frames using a custom-trained deep learning model. The goal is to demonstrate practical computer vision techniques for sports analytics and video understanding.
